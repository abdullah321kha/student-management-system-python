# Student Record System  
*A Python-Based CLI Management Project*

---

## Project Overview

The Student Record System is a command-line based Python application developed to manage student information efficiently.

This project demonstrates the implementation of core programming concepts including:

- Functions
- Loops
- Conditional Statements
- Lists & Dictionaries
- CRUD Operations

The system allows users to add, display, search, update, and delete student records through an interactive menu-driven interface.

# Features

## Add Student

This function allows users to insert new student records into the system.

### Included Information:
- Roll Number
- Student Name
- MIS ID
- Age
- Department
- Semester
- Section

![Add Student](images/add_student.png)

---

## Display Students

Displays all stored student records in a structured format.

### Functionality:
- Shows complete student details
- Loops through all records
- Handles empty record conditions

![Display Students](images/display_students.png)

---

## Search Student

Searches for a specific student using their Roll Number.

### Functionality:
- Finds matching records
- Displays student details instantly
- Handles missing records

![Search Student](images/search_student.png)

---

## Update Student

Updates an existing student’s information.

### Editable Fields:
- Name
- MIS ID
- Age
- Department
- Semester
- Section

![Update Student](images/update_student.png)

---

## Delete Student

Removes student records permanently from the system.

### Functionality:
- Deletes selected record
- Confirms successful deletion
- Handles invalid roll numbers

![Delete Student](images/delete_student.png)

---

# Main Menu Interface

The application uses a menu-driven system for navigation.

```bash
1. Add Student
```
